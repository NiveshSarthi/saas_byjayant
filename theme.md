# Synditech Website Theme Documentation

## Overview
The Synditech website employs a modern, dark-themed design with a futuristic tech aesthetic. It uses a black background with orange accent colors to create a sleek, professional look suitable for a software development company.

## Color Palette
- **Primary Background**: Black (#000)
- **Text Color**: White (#fff) with gray variations (gray-300, gray-400)
- **Accent Color**: Orange gradients
  - Primary Orange: #f97316 (orange-500)
  - Darker Orange: #ea580c (orange-600)
  - Deep Orange: #c2410c (orange-700)
- **Secondary Accents**: Blue in gradients (blue-900/20)

## Typography
- **Body Font**: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', etc.)
- **Heading Font**: Orbitron (imported from Google Fonts) - used for bold, futuristic headings
- **Text Gradient**: Applied to key text elements using orange gradient

## Layout and Components
- **Background**: Full-screen video background with overlay gradients for depth
- **Cards**: Subtle orange gradient backgrounds with borders
- **Buttons**:
  - Primary: Orange gradient background with rounded corners
  - Outline: Orange border with hover effects
- **Scrollbar**: Custom orange gradient scrollbar

## Animations
- **Fade-in**: Elements fade in with upward translation
- **Slide-in-left/right**: Content slides in from sides
- **Hover Effects**: Scale transforms on buttons and cards

## Key Design Elements
- Dark theme for modern, tech-focused appearance
- Orange accents for energy and innovation
- Video backgrounds for dynamic feel
- Grid-based layouts with responsive design
- Custom scrollbar and scrollbar styling

## Usage Guidelines
- Use `text-gradient` class for highlighted text
- Apply `btn-primary` and `btn-outline` for consistent button styling
- Utilize animation classes for smooth transitions
- Maintain black background with white text for readability
- Incorporate orange accents sparingly for emphasis
